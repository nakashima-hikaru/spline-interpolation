<h1 align="center">Spline Interpolation</h1>
This is a pure Rust implementation of spline interpolation.

## Feature

At the present, this project provides:

* (General) Spline
* Catmull-Rom Spline

### Cargo Feature Flags

* `decimal`: Add support for `rust-decimal::Decimal`
